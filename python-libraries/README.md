# CLI Movie Data Entry

## What this program does

This Python program lets users enter movie information in the terminal.
It shows example movies and allows users to add their own movie data.

## Requirements

Python 3 and the Rich library.

Install Rich:

pip install rich

## How to run

1. Open the terminal.
2. Go to the folder:

cd python-libraries

3. Run the script:

python cli_data_entry.py

## What happens

The program will ask you to enter:

* Movie title
* Release year
* Box office earnings

It will ask you to confirm the data.
You can add multiple movies.

## Output

The movie data will be saved in a file called:

movies_data.json
